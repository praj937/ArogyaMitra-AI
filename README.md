
# ArogyaMitra AI 🩺
**NHA Auto-Adjudication Hackathon 2026**

ArogyaMitra is an AI-driven ecosystem designed for clinical compliance and document integrity.

### 🎯 Problem Statements Addressed
1. **Clinical Document Classification (PS1):** Converting raw patient triage into structured, STG-compliant clinical summaries.
2. **Document Forgery Detection (PS3):** Using digital forensics (ELA) to identify tampered medical records.

### 🛠️ Technical Implementation
- **AI Engine:** Google Gemini 1.5 Flash
- **Frontend:** Streamlit (Python)
- **Forensics:** OpenCV & Pillow (Error Level Analysis)

### 🚀 How to Run
1. Clone the repo: `git clone https://github.com/praj937/ArogyaMitra-AI.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
